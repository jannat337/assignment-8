# SunCart ☀️ - Summer Essentials Store

## Purpose
A modern summer eCommerce platform where users can explore and purchase seasonal products like sunglasses, summer outfits, skincare, beach accessories, and more.

## Live URL
🔗 [https://assignment-8-1ncj.onrender.com](https://assignment-8-1ncj.onrender.com)

## Key Features
- 🌅 Hero Section with Image Slider
- 🔥 Popular Products Section
- 🛍️ All Products Page
- 🔒 Protected Product Details Page (Login Required)
- 🔐 User Authentication (Email/Password & Google)
- 👤 My Profile Page
- ✏️ Update Profile Information
- ☀️ Summer Care Tips Section
- 🏷️ Top Brands Section
- 📱 Fully Responsive (Mobile, Tablet, Desktop)

## Tech Stack
- **Next.js** - React Framework
- **Tailwind CSS** - Styling
- **DaisyUI** - UI Components
- **BetterAuth** - Authentication

## NPM Packages Used
- `better-auth` - Authentication
- `better-sqlite3` - Database
- `animate.css` - Animations
- `react-icons` - Icons
- `tailwindcss` - CSS Framework
- `daisyui` - UI Components
